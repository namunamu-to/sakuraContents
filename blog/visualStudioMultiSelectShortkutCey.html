[Visual Studio]個人的に便利でよく使うショートカットキー一覧を紹介

まえがき

私がvisual studioを使う時、ショートカットキーを調べようとしても検索結果がvscodeで埋められてしまい、調べるのが少し面倒だと感じました。

ですので、似たようなことを考えている方が少しでもショートカットキーについて調べやすくなってほしいと思い、この記事を作成するに至りました。

特に、vscodeでいうctrl+dの複数選択のショートカットキーは探すのが難しいと思います。

この記事ではカーソル操作の項目で、vscodeでいうctrl+dの複数選択のショートカットキーも紹介しているので、参考にしていただければ幸いです。

ファイル操作

新規作成 : Ctrl + N

開く : Ctrl + O

上書き保存 : Ctrl + S

名前を付けて保存 : Ctrl + Shift + S

閉じる : Ctrl + F4

カーソル操作

行入れ替え : Alt + 上下キー

次の行に新規行を挿入してカーソル移動 : Ctrl + Shift + Enter

前の行に新規行を挿入してカーソル移動 : Ctrl + Enter

複数選択 : Shift + Alt + .

編集

切り取り : Ctrl + X

コピー : Ctrl + C

貼り付け : Ctrl + V

元に戻す : Ctrl + Z

やり直す : Ctrl + Y

置換 : Ctrl + H

ナビゲーション

次のタブに移動 : Ctrl + Tab

前のタブに移動 : Ctrl + Shift + Tab

現在の行の先頭に移動 : Home

現在の行の末尾に移動 : End

次の単語に移動 : Ctrl + Right

前の単語に移動 : Ctrl + Left

検索

ファイル内で検索 : Ctrl + F

プロジェクト内で検索 : Ctrl + Shift + F

置換 : Ctrl + H

Find All References : Shift + F12

デバッグ

デバッグの開始 : F5

デバッグの停止 : Shift + F5

ステップイン : F11

ステップオーバー : F10

実行へ移動 : Ctrl + F10

ブレークポイントの追加/削除 : F9

リファクタリング

リファクタリング : Ctrl + R

リネーム : Ctrl + R + R

抽出 : Ctrl + R + E

インライン化 : Ctrl + R + I

その他

ヘルプを表示 : F1

コマンドパレットを表示 : Ctrl + Shift + P

環境設定を開く : Ctrl + ,

全画面表示 : F11

ナビゲーションバーを表示/非表示 : Ctrl + Tab
